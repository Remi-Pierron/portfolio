Projet mené en groupe de 3 - novembre 2024

Concevoir un site web dynamique relié à une base de données.

Quel était l'objectif de notre travail ?

Nous devions réaliser un site web dynamique avec des graphiques interactifs alimentés par une base de données. Le thème était libre.

Comment avons-nous procédé ?

Nous avons choisi le thème de l'écologie, et nous avons décidé de faire plusieurs page avec plusieures approches différentes. Une partie des pages est dédiée à la présentation de graphiques et de tableaux, tandis qu'une autre partie du site est dédiée à l'intéractivité entre les utilisateurs, notamment grâce à un forum de discussion, un blog ainsi qu'une page présentant la météo actuelle grâce à une API. Nous avons aussi développé tout un backend permettant de gérer le chargement des données dans les tables de la base en Python.

Quelles difficultés avons-nous rencontrées ?

La plus grandes difficulté a été de rendre le site adapté à toutes les tailles d'écrans.

Quelle note avons-nous obtenue ?

Notre travail nous a valu la note de 15/20.

Quelles sont mes compétences acquises ?

Ce projet m'a permis d'améliorer mes compétences en pogrammation web et m'a apporté une compréhension du fonctionnement d'une base de données en arrière plan d'un site.
