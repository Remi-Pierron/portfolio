SQL Oracle Looping
Projet en groupe de 2 - Novembre 2024

Construire, alimenter et exploiter une base SQL Oracle

Quel était l'objectif de notre travail ?

Nous avons été chargé de concevoir une base de données pour une société fictive, en utilisant SQL et Oracle. Nous devions créer les tables, les relations entre elles, ainsi que les requêtes nécessaires pour exploiter les données. Nous avons conçu le MLD et le MCD avec Looping.

Comment avons-nous procédé ?

Nous avons commencé par analyser les besoins de la société fictive, puis nous avons conçu le MCD et le MLD. Ensuite, nous avons créé les tables dans Oracle et alimenté la base de données avec des données fictives. Enfin, nous avons écrit des requêtes SQL pour extraire et manipuler les données.

Quelles difficultés ai-je rencontrées ?

Nous avons rencontré des difficultés lors de la création des relations entre les tables, car il était important de bien comprendre les cardinalités.

Quelle note ai-je obtenue ?

Nous avons obtenu la note de 16.8/20

Quelles sont mes compétences acquises ?

J'ai acquis des compétences en conception de bases de données, en SQL et en utilisation d'Oracle.
