Projet mené en groupe de 4 - Mars 2025

Concevoir un site web dynamique avec les données AirBNB de la ville de Paris

Quel était l'objectif de notre travail ?

Nous devions créer un site web dynamique permettant de visualiser les données AirBNB de la ville de Paris. Le site devait être interactif et permettre aux utilisateurs d'explorer les données de manière intuitive. Nous avons utilisé des bibliothèques JavaScript telles que Plotly pour créer des graphiques interactifs et rendre le site plus attrayant.

Comment avons-nous procédé ?

Nous avons commencé par récupérer les données AirBNB de la ville de Paris. Ensuite, nous avons utilisé JavaScript pour manipuler et afficher ces données sur le site web. Nous avons intégré des graphiques interactifs à l'aide de la bibliothèque Plotly, ce qui a permis aux utilisateurs d'explorer les données de manière visuelle. Enfin, nous avons hébergé le site sur GitHub pour le rendre accessible au public.

Quelles difficultés avons-nous rencontrées ?

La principale difficulté était que nous devions respecter un délai de conception très court. Nous avons donc dû travailler rapidement pour terminer le projet dans les temps.

Quelle note avons-nous obtenue ?

Notre travail nous a valu la note de 15/20.

Quelles sont mes compétences acquises ?

J'ai acquis des compétences en développement web dynamique avec JavaScript, en création de graphiques interactifs avec Plotly et en hébergement de projets sur GitHub.
