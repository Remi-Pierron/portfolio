Projet mené en groupe de 4 - février 2025

Concevoir un logiciel permettant de mettre en place une API à partir d'un jeu de données volumineux

Quel était l'objectif de notre travail ?

L'objectif de ce projet était de créer un entrepôt de données à partir d'un ensemble de données volumineux et hétérogène. Nous avons utilisé Python pour le nettoyage et la transformation des données, puis nous avons intégré ces données dans un entrepôt de données en utilisant Docker Desktop. Enfin, nous avons créé des visualisations interactives avec Power BI pour analyser les données.

Comment avons-nous procédé ?

Nous avons commencé par explorer les données brutes pour comprendre leur structure et identifier les problèmes de qualité. Ensuite, nous avons utilisé Python pour nettoyer et transformer les données, en supprimant les doublons, en corrigeant les erreurs et en normalisant les formats. Une fois les données prêtes, nous avons utilisé Docker Desktop pour créer un environnement d'entrepôt de données et y intégrer nos données nettoyées. Enfin, nous avons utilisé Power BI pour créer des tableaux de bord interactifs permettant d'analyser les données de manière visuelle.

Quelles difficultés avons-nous rencontrées ?

Nous avons rencontré plusieurs défis au cours de ce projet. L'un des principaux défis était de gérer la taille et la complexité des données, ce qui a nécessité une attention particulière lors du nettoyage et de la transformation. De plus, l'intégration des données dans Docker Desktop a présenté des défis techniques, notamment en ce qui concerne la configuration de l'environnement. Enfin, la création de visualisations interactives avec Power BI a nécessité un apprentissage approfondi de l'outil pour tirer le meilleur parti de ses fonctionnalités.

Quelle note avons-nous obtenue ?

Notre travail nous a valu la note de 17,5/20.

Quelles sont mes compétences acquises ?

Grâce à ce projet, j'ai acquis des compétences solides en nettoyage et transformation de données avec Python, ainsi qu'une compréhension approfondie de la création d'entrepôts de données avec Docker Desktop. J'ai également développé mes compétences en visualisation de données avec Power BI, ce qui m'a permis de créer des tableaux de bord interactifs et informatifs. Enfin, j'ai renforcé mes compétences en travail d'équipe et en gestion de projet.
